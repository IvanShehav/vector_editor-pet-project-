# Векторный редактор
### Итоги работы над проектом:

1. **Завершены все этапы проекта:**
    * Реализованы все 8 модулей основной программы.
    * Выполнены все сопутствующие дополнительные задания к модулям.

2. **Реализованы дополнительные функции:**
    
    a. **Выделение области (Rubber Band Selection):** добавлена возможность выделения нескольких фигур рамкой при зажатой клавише `Shift` в режиме инструмента `Select`.
    
    b. **Горячие клавиши (Hotkeys):** быстрый возврат к инструменту выделения (`Select`) по нажатию клавиши `Space` (Пробел).
    
    c. **Инструмент «Многоугольник» (Polygon Tool):** добавлен и адаптирован новый инструмент Polygon, который может замыкаться, если имеет более 2 вершин и последняя попадает в первую или стать любой ломаной, если по завершению нажать Enter
   
**ShortCut:**
1. Ctrl + Q (выйти)
2. Ctrl + S (сохранить)
3. Ctrl + O (открыть)
4. Space (выбор инструмента Select)
5. Ctrl + G (объединение в группу)
6. Ctrl + U (разъединение группы)
7. Ctrl + Z (отмена последнего действия)
8. Ctrl + Y (восстановить отмену)
9. Enter при выбранном Polygon (завершение создание фигуры)
10. Зажатый Shift при выбранном Select (создание области выделения)
11. Зажатый Ctrl при выборе Select (множественное выделение)
12. Стрелочка влево/вправо/вверх/вниз (смещение на 1 пиксель в соответствующем направлении)
13. Стрелочка влево/вправо/вверх/вниз + зажатый Shift (смещение на 10 пикселей в соответствующем направлении)
14. Delete (удаление выбранного (-ых) объекта (-ов)

**Создание .exe файла в консоле**
1. ```pip install pyinstaller```
2. ```pyinstaller --noconfirm --onefile --windowed --name "VectorEditor" main.py```
